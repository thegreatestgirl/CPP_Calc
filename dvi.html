CPP SmartCalc_v1.0

About SmartCalc_v1.0

This program is my analog of popular app: calculator. It contains unary plus and minus, all basic arithmetic operations: plus, minus, multiply, divide. Also, this calculator supports some math functions: sqrt, ln (natural logarithm), log (decimal logarithm). And SmartCalc_v1.0 keeps trigonometric functions: cos(), sin(), tan(), acos(), asin(), atan(). I also added Pi const (p at app) and exponent form (for exm. 3e+8).

Install of SmartCalc_v1.0

The app can be installed by make install command. This action will create a folder on your desktop with an app in it. The Calc app starts by double clicking on it.

Uninstall of SmartCalc_v1.0

The app can be uninstalled by make uninstall command. This action will remove created folder on your desktop.
